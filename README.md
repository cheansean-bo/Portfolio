# Cheansean Bo Portfolio

Hi, this is Sean! 
Welcome to my [portfolio](https://cheansean-bo.github.io/Portfolio/), the site was created by me using ReactJS. 

# Technologies
- ReactJS
- CSS

# Prerequisites
- node.js
- npm
- git

# Installation and Setup
1. Clone the repositories
   - git clone https://github.com/cheansean-bo/Portfolio
   - cd Portfolio
2. Install Dependencies
   - npm install
3. Run the development server
   - npm run start
   - Open in browser http://localhost:5173
